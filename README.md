# SCT_SD_1 - Temperature Converter

## Task Description
Create a program that converts temperatures between Celsius, Fahrenheit, and Kelvin scales.

## Features
- Converts between all three temperature scales:
  - Celsius ↔ Fahrenheit
  - Celsius ↔ Kelvin
  - Fahrenheit ↔ Kelvin
- User-friendly command-line interface.
